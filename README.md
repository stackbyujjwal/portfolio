### 🚀 The Story Behind The Code
This repository holds my classic, "Version 1" personal portfolio. Built a few years ago, it represents the foundational steps of my journey as a web developer. 

While the data inside reflects an earlier timeline (rating my JavaScript at 25% and my age at 21!), the **UI/UX architecture, dark mode aesthetics, and clean CSS layouts** stand the test of time. It’s a testament to my early focus on design before diving deep into complex backend logics, databases, and C++ architectures.

*P.S. Yes, there are a few legacy typos like "Contect Me" and a testimonial from myself—I keep them there as a badge of honor to show how much my debugging skills (and self-confidence) have grown!* 😉

---

### ✨ Classic Features
- **The "V1" Dark Mode:** A sleek, premium dark UI with vibrant purple (`#8A2BE2`) accents.
- **Responsive Design:** Custom-built layouts that look great on any screen.
